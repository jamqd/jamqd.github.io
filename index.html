Perspective

Perspective is a web app that helps users find different perspectives on any search query. In a world of increasing political polarization, listening to and understanding the other side of an issue is more important than ever. This EV Hacks Most Useful Hack Winnig project aims to help solve this issue.

Technologies Used: Java, HTML, Bing News Search API, newsapi.org API

LeNet 5 Implementation

This is an implementation of the LeNet paper in TensorFlow. This is part of my exploration in reading and implementing research papers. Currently I'm working on implementing a TensorFlow JS web app that can recognize digits drawn by a mouse in the browser.

Originally this model was trained on Google Colab, but now I train models on the gaming PC I built that is now repurposed as a remote Jupyter Lab Server back at home. This GPU server has a Nvidia GTX 1070.

Technologies Used: Python, Tensorflow, Google Colab, Jupyter Lab/Notebook
